# HOSTİNGER WEB SİTE KURULUM REHBERİ

## ADIM 1: FRONTEND DOSYALARINI HAZIRLA

1. Bu rehberdeki 'frontend-build.zip' dosyasını indirin
2. Zip dosyasını çıkarın
3. İçindeki dosyaları göreceksiniz

## ADIM 2: HOSTİNGER'E YÜKLEYİN

1. hPanel'e giriş yapın
2. 'File Manager' tıklayın
3. 'public_html' klasörüne girin
4. Tüm dosyaları sürükle bırak yapın

## ADIM 3: BACKEND'İ VERCEL'E KURUN

1. Vercel.com'a gidin
2. GitHub ile giriş yapın
3. Proje dosyalarını yükleyin

## SONUÇ

Web siteniz taxiturlihof.com adresinde çalışacak!
